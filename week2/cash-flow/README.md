# Cash Flow Tracker

A salary and expense tracking dashboard built with vanilla JavaScript.

---

## Features

- Add salary and expenses
- Real-time balance calculation
- LocalStorage support
- Chart visualization
- PDF export
- Budget alert

---

## Tech Used

HTML, CSS, JavaScript, Chart.js, jsPDF

---

## How to Run

```bash
cd cash-flow
npx serve .
```

Then open `http://localhost:3000` in your browser.

Or just double-click `index.html` to open directly in your browser.

> Note: Currency converter requires internet connection (uses Frankfurter API).
